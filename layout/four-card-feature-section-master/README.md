# Four Card Feature Section

![Design preview for the Four Card Feature Section coding challenge](./design/desktop-preview.jpg)

## Project Description

This project is a responsive "Four Card Feature Section" built using HTML and CSS, following a Frontend Mentor challenge design. The goal was to replicate the design as closely as possible while practicing layout, responsiveness, and accessibility.

## Features

- Fully responsive design (desktop & mobile)
- Flexbox layout for the main container
- CSS Grid for grouping multiple cards
- Reusable CSS variables for colors, fonts, and weights
- Semantic HTML with accessible alt text
- Shadow and border styling to match the design
