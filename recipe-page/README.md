## Overview

This project is a recipe page that displays:

- Dish image
- Preparation time
- Ingredients
- Step-by-step instructions
- Nutrition information

It was built mobile-first and adapts to desktop screens gracefully.

---

## Features

- **Mobile-first design**: fully responsive below 375px.
- **Desktop breakpoint**: scales layout at `min-width: 786px`.
- **Accessible**: screen-reader-only heading, semantic HTML.
- **Responsive table**: nutrition table adapts to small and large screens.
- **Clean BEM-style CSS**: consistent naming for easy maintenance.
- **Custom fonts and color palette**: matches the design mockup.

---

## Technologies

- HTML5
- CSS3 (Variables, Flexbox, Media Queries)
- Google Fonts (`Outfit` & `Young Serif`)
