# Frontend Mentor - Blog Preview Card

![Design preview for the Blog Preview Card coding challenge](./preview.jpg)

## Overview

This project is a solution to the [Frontend Mentor Blog Preview Card challenge](https://www.frontendmentor.io/challenges/blog-preview-card-component-7KlnnDR5).

The goal of this challenge was to build a **responsive blog preview card** using **HTML and CSS** that closely matches the provided design for mobile and desktop screens.

---

## Built With

- HTML5
- CSS3 (including CSS variables and Flexbox/Grid)
- Google Fonts (Figtree)
- Responsive Design techniques

---

## Features

- Fully **responsive** across mobile and desktop devices
- **Centered card** with fluid width
- **Hover effects** on interactive elements (headings)
- Clean and semantic HTML structure
- Modern CSS techniques:
  - CSS variables for colors
  - Flexbox and Grid for layout
  - `clamp()` for fluid font sizing and spacing
- Images optimized and styled with `object-fit: cover` for consistency
