# Frontend Mentor - Social Links Profile

![Design preview for the Social Links Profile coding challenge](./preview.jpg)

## Overview üëã

This is a **responsive, mobile-first social links profile card** built using **HTML and CSS**, based on the [Frontend Mentor challenge](https://www.frontendmentor.io/challenges/social-links-profile-card-component-8v1fZMi0q).

The goal was to replicate the design as closely as possible while keeping it **accessible, responsive, and maintainable**.

---

## üñåÔ∏è Design & Style Guide

- **Mobile-first** layout: Designed to work on all devices from 320px up.
- **Colors**:
  - Green: `hsl(75, 94%, 57%)`
  - White: `hsl(0, 0%, 100%)`
  - Grey 700: `hsl(0, 0%, 20%)`
  - Grey 800: `hsl(0, 0%, 12%)`
  - Grey 900: `hsl(0, 0%, 8%)`
- **Typography**:
  - Font family: [Inter](https://fonts.google.com/specimen/Inter)
  - Font weights: 400 (regular), 600 (medium), 700 (bold)
  - Base font size: 14px (as per style guide)
- **Accessibility**:
  - Proper semantic HTML: `<main>`, `<section>`, `<nav>`
  - `.sr-only` for hidden heading
  - Focus-visible states for interactive elements
  - Good color contrast

---

## üîß Features

- **Mobile-first and responsive**  
  Layout adjusts seamlessly from 320px mobile screens up to desktop widths.
- **Flexbox layout** for clean, centered alignment.
- **Hover & focus states** on all social buttons for interactivity.
- **CSS variables** for colors and font weights, making the code maintainable.
- **Avatar with circular crop** and proportional spacing.
- Works well on landscape devices and unusual viewports (e.g., Nest Hub).

---

## üíª Tech Stack

- **HTML5**
